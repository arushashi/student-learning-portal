# System Design Learning Portal

A comprehensive student learning platform for mastering system design concepts with interactive quizzes, visual diagrams, and interview preparation materials.

## 🚀 Features

- **Interactive Index Page**: Clean navigation hub with search functionality
- **Comprehensive Topics**: 15+ system design concepts with detailed explanations
- **Structured Learning**: Each topic includes definition, explanation, key points, interview Q&A, and quizzes
- **Visual Learning**: Diagrams and flowcharts for better understanding
- **Interactive Quizzes**: Test knowledge with instant feedback
- **Student-Friendly UI**: Clean, modern design optimized for learning
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Progress Tracking**: Monitor learning progress through topics
- **Bookmark System**: Save favorite topics for quick access
- **Dark Mode**: Toggle between light and dark themes

## 🚀 Getting Started

1. **Clone or Download**: Get the project files to your local machine
2. **Open in Browser**: Navigate to the project folder and open `index.html`
3. **Start Learning**: Browse topics and begin your learning journey

### No Setup Required!
This is a static website that runs entirely in the browser. No installation, database, or server setup needed.

## 📚 How to Use

1. **Start at Homepage**: Browse available topics on the index page
2. **Search Topics**: Use the search bar to find specific concepts
3. **Select a Topic**: Click on any topic card to start learning
4. **Follow the Flow**: Progress through definition → explanation → key points → interview Q&A → quiz
5. **Test Knowledge**: Complete quizzes to reinforce learning
6. **Track Progress**: Monitor your completion status for each topic
7. **Bookmark Favorites**: Save important topics for quick reference

## 🔧 Customization

### Adding New Topics
1. Copy the concept template from `components/concept-template.html`
2. Replace placeholder content with your topic information
3. Save as new HTML file in the `pages/` directory
4. Add topic card to `index.html`
5. Update navigation and links as needed

### Styling Customization
- Modify colors and themes in `assets/css/styles.css`
- Adjust concept page layouts in `assets/css/concept.css`
- Add custom animations and transitions

### Functionality Enhancement
- Extend JavaScript features in `assets/js/script.js`
- Add new interactions in `assets/js/concept.js`
