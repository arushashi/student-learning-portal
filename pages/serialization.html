Serialization

Data exchange formats for communication between systems

Intermediate ⏱ 18 min

Definition

Serialization is the process of converting data structures or objects into a format that can be stored or transmitted and reconstructed later.

Common Formats

  • JSON: Lightweight, human-readable, widely used
  • XML: Structured, self-describing, verbose
  • Protocol Buffers: Binary, efficient, schema-based
  • MessagePack: Binary, compact, fast
  • Avro: Schema evolution support, binary

Use Cases

  • API request/response data
  • Database storage
  • Message queue payloads
  • Configuration files
  • Cache storage
