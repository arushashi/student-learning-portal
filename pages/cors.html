CORS

Cross-Origin Resource Sharing for secure web communication

Definition

What is CORS?

CORS is a security mechanism that allows web applications to make requests to different domains while maintaining security.

Key Insight: Think of CORS like a bouncer at a club - it checks IDs (origins) before letting requests through.

Simple Explanation

Understanding CORS

CORS allows servers to specify which origins can access their resources, preventing malicious cross-origin requests.

How CORS Works

  • Simple Requests: GET, POST, HEAD with basic headers
  • Preflight Requests: OPTIONS method before actual request
  • Headers: Access-Control-Allow-Origin, etc.
  • Credentials: Cookies and authentication

Common CORS Headers

  • Access-Control-Allow-Origin
  • Access-Control-Allow-Methods
  • Access-Control-Allow-Headers
  • Access-Control-Max-Age

Real-World Example

Browser Security

Browsers enforce CORS to prevent malicious scripts from accessing sensitive data across different origins.

Key Points & Summary

Security

Prevents malicious cross-origin requests and data theft.

Headers

Uses HTTP headers to control cross-origin access.

Preflight

OPTIONS requests check permissions before actual requests.

Credentials

Controls whether cookies and auth can be included.

Quick Summary

  • CORS enables secure cross-origin requests
  • Server-side configuration controls access
  • Browsers enforce CORS automatically
  • Essential for modern web applications

Interview Questions & Answers

What is CORS and why is it needed?

CORS (Cross-Origin Resource Sharing) is a security mechanism that allows web applications to make requests to different domains. It's needed to prevent malicious scripts from accessing sensitive data while enabling legitimate cross-origin requests.

What is a preflight request?

A preflight request is an OPTIONS request sent by the browser before the actual request to check if the cross-origin request is allowed. It's used for requests that aren't simple (like those with custom headers or methods other than GET/POST).
